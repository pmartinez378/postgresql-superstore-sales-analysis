# PostgreSQL Superstore Sales Analysis

## Project Overview
This project analyzes retail sales data using PostgreSQL. The dataset contains transactional sales records including products, regions, discounts, and profitability.

The objective of the analysis is to extract business insights and identify trends in sales performance, profitability, and discount strategies.

## Tools Used
- PostgreSQL
- pgAdmin 4
- SQL

## Dataset
Superstore retail dataset containing:

- Orders
- Products
- Sales
- Profit
- Discounts
- Regional data

## Key Business Questions
This project answers several analytical questions:

1. What is the total sales and profit generated by the business?
2. Which regions generate the most revenue?
3. Which product categories are most profitable?
4. How do discounts impact profitability?
5. Which products generate the highest profit?

## Key Insights
- Certain regions consistently outperform others in total revenue.
- Heavy discounting often correlates with lower profit margins.
- Some product categories contribute disproportionately to total profit.
- A small subset of products drives a significant portion of profits.

## Example SQL Query

```sql
SELECT region,
       SUM(sales) AS total_sales
FROM superstore
GROUP BY region
ORDER BY total_sales DESC;
